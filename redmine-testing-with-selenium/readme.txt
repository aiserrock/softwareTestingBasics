Необходимо составить тест-кейсы для тестирования функций системы управления проектами Redmine, развёрнутой по адресу https://yar.fruct.org/

Тестированию подлежат следующие возможности системы:

Переход к списку проектов по ссылке Projects с главной страницы.
Авторизация пользователя.
Видимость закрытого проекта для авторизированного пользователя.
Добавление задачи в трекер по кнопке New issue со страницы со списком задач.
Переход к странице добавления задачи из контекстного меню.
Редактирование задачи.
